# TOAB 系统入侵解密游戏框架

一个基于 Web 的模拟系统入侵解密游戏框架，提供完整的文件系统、终端、解密工具等功能。

├── css/
│   ├── main.css           # 全局样式和登录页面样式
│   ├── homepage.css       # 官网首页样式

## 功能特性

### Phase 1: 系统基础（已完成）

- ✅ **官网首页**：TOAB 系统官网，隐藏解密入口
- ✅ **解密系统**：通过隐藏线索触发解密，获得初始账户密码
- ✅ **登录页面**：使用解密获得的凭证登录系统
- ✅ **系统主界面**：包含仪表板、文件系统、终端、解密工具、任务日志等模块
- ✅ **文件系统模拟**：完整的目录结构和文件浏览功能
- ✅ **终端系统**：支持基本命令（ls, cd, pwd, help, clear）
- ✅ **代码解密工具**：支持 Base64、Caesar、Reverse、Hex 等多种解密方法
- ✅ **任务日志系统**：任务管理和进度跟踪
- ✅ **本地存储**：游戏进度保存和用户信息管理

## 使用方法

### 游戏流程

1. **访问官网首页**
   - 在浏览器中打开 `index.html`
   - 浏览 TOAB 系统官网，寻找隐藏的解密入口

2. **触发解密系统**
   - **方法一**：在页脚右下角区域连续点击 5 次
   - **方法二**：使用键盘快捷键 `Ctrl + Shift + D`
   - **方法三**：查看页面源代码，寻找隐藏线索
   - 解密面板会出现，需要输入正确的访问密钥

3. **获取登录凭证**
   - 有效的访问密钥：`TOAB2025SECURE`（隐藏在新闻动态中）
   - 解密成功后会显示初始账户的用户名和密码
   - 点击"前往登录页面"按钮

4. **登录系统**
   - 在登录页面输入从官网解密获得的用户名和密码
   - 登录成功后进入系统主界面

2. **系统主界面**
   - **仪表板**：查看系统状态、任务数量、解密进度等
   - **文件系统**：浏览和访问系统中的文件
   - **终端**：使用命令行操作文件系统
   - **解密工具**：解密加密的代码和文件
   - **任务日志**：查看任务进度和故事线

3. **终端命令**
   - `help` - 显示帮助信息
   - `ls` - 列出当前目录文件
   - `cd <目录>` - 切换目录
   - `pwd` - 显示当前路径
   - `clear` - 清空终端

4. **解密工具**
   - 在解密工具页面输入加密代码
   - 系统会自动尝试多种解密方法
   - 解密结果会显示在输出区域
